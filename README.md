# Project-Behavioral-Decision-Analytics
“I built a behavioral analytics system with a custom risk model”
🚀 Behavioral Decision Analytics
📌 Overview

This project analyzes human behavioral patterns using real-world Twitter data to understand how sentiment changes over time and how it influences decision-making.

The goal is to identify high-risk time periods where users are more likely to express negative emotions and potentially make poor decisions.

🎯 Problem Statement

Do people express more negative emotions at certain times of the day?
Can we identify high-risk decision windows using sentiment patterns?

📊 Dataset
Source: Airline Twitter Sentiment Dataset
Total Records: 14,000+ tweets
Features Used:
Text
Sentiment
Timestamp

🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib)
NLP (Sentiment Analysis)
Microsoft Power BI Desktop
Jupyter Notebook

⚙️ Project Workflow
Data Cleaning & Preprocessing
Sentiment Encoding (Positive, Neutral, Negative)
Time Feature Extraction (Hour, Time Zone)
Behavioral Pattern Analysis
Custom Risk Score Engineering
Dashboard Visualization

📈 Key Features
⏱️ Sentiment vs Time Analysis
🌙 Time Zone-Based Behavior Insights
⚠️ Decision Risk Score (Custom Feature)
📊 Interactive Power BI Dashboard

🧠 Key Insights
Negative sentiment peaks during late-night hours (10 PM – 3 AM)
Users show higher emotional volatility at night
Tweet activity remains high during negative sentiment periods
A custom Decision Risk Score highlights night as a high-risk window

💼 Business Impact
Helps identify high-risk behavioral periods
